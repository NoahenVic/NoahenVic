---

## Featured projects

### [BASE985161](https://github.com/NoahenVic/BASE985161)
A binary-to-text codec using a huge Unicode alphabet of **985,161 symbols**. It includes a one-page web app, Python support and JavaScript support.

### [BASE262144](https://github.com/NoahenVic/BASE262144)
An experimental Unicode encoding format where each character represents **18 bits**. Strange output, fully reversible system.

### [RC-CUP-Template](https://github.com/NoahenVic/RC-CUP-Template)
A clean site template built for RC Cup teams, with JSON-driven content, NL/EN support and a generator workflow.
